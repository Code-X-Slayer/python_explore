# Python Explore

A personal repository documenting my journey to become a better Python developer.

The goal wasn't to **master Python**—there's always more to learn. Instead, I wanted to build a strong foundation by exploring the language in depth, writing code every day, and building projects alongside tutorials instead of just watching them.

This repository contains the scripts, experiments, notes, and practice code I wrote while following a condensed Python roadmap based primarily on **Corey Schafer's** tutorials. Every concept was practiced immediately through code, and each stage ended with a mini project to reinforce what I learned.

## Learning Roadmap

- ✅ Stage 1 – Python Fundamentals
- ✅ Stage 2 – Intermediate Python
- ✅ Stage 3 – Files & Real Data
- ✅ Stage 4 – Pythonic Python
- ✅ Stage 5 – Object-Oriented Programming
- ✅ Stage 6 – Working with APIs
- ✅ Stage 7 – Modern Python
- ✅ Stage 8 – AsyncIO & Pydantic
- ⏳ Stage 9 – FastAPI (In Progress)
